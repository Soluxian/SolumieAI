# META-HEARTBEAT (Weekly Self-Optimization)

Run every Sunday 6pm EST.

## Tasks
1. Review all cron jobs for redundancy
2. Check token usage patterns per job
3. Identify jobs that never fire (frequency mismatches)
4. Optimize intervals based on actual usage
5. Document changes to HEARTBEAT.md
6. Archive removed jobs to HEARTBEAT-ARCHIVE.md

## Optimization Criteria
- Jobs under-utilized → Reduce frequency or remove
- Jobs overlapping → Consolidate
- High token-burn jobs → Move to Ollama or reduce
- Personal growth jobs → Protected, never reduce
